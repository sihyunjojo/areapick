# 상추:창고 <br>상권 추천 창업 레츠고
SSAFY 11기 특화 프로젝트 

### 프로젝트 기간
2024.08.19 ~ 2024.10.11 (8주)

### 프로젝트 소개
상추:창고는 프랜차이즈 창업을 희망하는 사람들에게 원하는 지역 상권 및 프랜차이즈에 대한 다양한 정보와 분석을 제공하여 성공적인 창업의 길을 더욱 쉽게 열어갈 수 있습니다.
